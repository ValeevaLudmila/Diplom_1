## ⚡ Запуск миссии

1️⃣ Установить зависимости:

```bash
pip install -r requirements.txt
```

2️⃣ Запустить все тесты:

```bash
pytest -v
```

3️⃣ Проверить покрытие кода:

```bash
pytest --cov=praktikum --cov-report=html
```

4️⃣ Просмотреть Allure отчет (если включен):

```bash
pytest --alluredir=allure-results
allure serve allure-results
```

## 🔭 Космические заметки (Баги / Known Issues)

* **Авторизация не работает** — Access Token не устанавливается в localStorage
* **Drag & Drop не работает** — ингредиенты не добавляются в корзину конструктора
* **Невозможно оформить заказ** — модальное окно с номером заказа не появляется

Все баги отмечены приоритетом: **High**.
Шаги воспроизведения доступны в `tests/conftest.py` и интеграционных тестах.
